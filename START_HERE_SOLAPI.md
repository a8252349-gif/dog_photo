# 처음 시작할 곳

1. `증멍사진_쿠폰관리_템플릿.xlsx`를 Google Drive에 업로드하고 Google 스프레드시트로 변환합니다.
2. `google-apps-script/CouponServer.gs` 전체를 스프레드시트의 Apps Script에 붙여넣습니다.
3. Apps Script 속성과 웹 앱 배포를 `google-apps-script/README.md` 순서대로 진행합니다.
4. Render 환경변수에 쿠폰 서버와 SOLAPI 정보를 입력합니다.
5. `SOLAPI_COUPON_SETUP.md`의 curl 명령으로 문자와 쿠폰 발급을 테스트합니다.
