# 라우트 및 로고 수정본

이 버전은 다음 문제를 수정합니다.

- `/dog-id-photo` 404 수정
- `/cat-id-photo` 404 수정
- `/faq` 404 수정
- `/manifest.webmanifest` 404 수정
- 제공된 `증멍사진` 로고를 헤더와 푸터에 적용

반드시 기존 `pet-id-photo-render-gpt-image-2` 폴더가 아니라 이 폴더에서 실행하세요.

```bash
npm install
npm run dev
```

확인 주소:

- http://localhost:3000/
- http://localhost:3000/dog-id-photo
- http://localhost:3000/cat-id-photo
- http://localhost:3000/faq
- http://localhost:3000/manifest.webmanifest
